DeleteModel

A Query class that allows for quickly deleting of models by ID.

Constructor

new DeleteModel(database, escaper, queryExecuter, model)

Initialize the Query instance.
Parameters:
Name Type Description
database Database The database to delete from.
escaper Escaper An instance of an Escaper matching the database type (e.g. MySQLEscaper).
queryExecuter QueryExecuter A QueryExecuter instance that implements the delete method.
model Object A model object to delete. Each key in the object should map to a table. The value associated with the key should be an object or an array of objects wherein each key maps to a column. The primary key is required for each model.
